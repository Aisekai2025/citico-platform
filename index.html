About CitiCo

CitiCoは、市民（Citizen）が企業（Company）と共に（Co-creation）社会課題を解決していくためのパブリック・プラットフォームです。

私たちは、日々の暮らしの中に埋もれている「小さな声」に耳を澄ませます。 TikTokなどのソーシャルメディアを通じて発信される生の声を起点に、食糧支援や教育支援につながるプロジェクトを立ち上げ、 持続可能な社会還元モデルを構築します。

How it works

市民の声がプロジェクトになり、社会へ還元されるまでのプロセス。

01

Voice

市民がTikTokに投稿。
ハッシュタグ #CitiCo をつけ、影だけや抽象的な表現を用いて、日常の「困りごと」や「アイデア」を発信します。

02

Pickup

CitiCo公式が投稿をピックアップ。
世界観を統一したCapCutテンプレートを用いてリメイクし、課題を可視化します。

03

Analysis

Geminiによる分析と企画。
投稿された課題の背景をAIが整理・言語化し、企業パートナーと共に具体的な解決策を企画します。

04

Return

収益を社会に還元。
プロジェクト収益の一部を食糧・教育支援へ還元。その実績はGitHub上で透明性を持って公開されます。

Tools & Roles

CitiCoのエコシステムを支える、4つの主要チャネル。

Gemini

市民の声（動画データ）から課題の本質を分析し、企画・言語化を担います。法的・倫理的なリスクチェックも行い、安全なプロジェクト進行を支えます。

TikTok

市民参加のメインステージです。ハッシュタグ #CitiCo のハブとして機能し、ここから全てのプロジェクトが始まります。

CapCut

CitiCoの世界観を保つための「工房」です。誰でも簡単に美しい発信ができるよう、統一された動画テンプレートを提供します。

GitHub

プラットフォームの心臓部です。プロジェクトのアーカイブ、ポリシーの策定、そして社会還元レポートをパブリックに公開します。

Social Return

得られた収益は、透明性を持って社会課題の解決へ充当します。
現在はパイロットフェーズであり、詳細なスキームは順次公開予定です。

Food Support
フードバンクや地域食堂への寄付、廃棄食材を活用した新製品開発による還元など、食のセーフティネット構築を支援します。
Education Support
経済的な理由で学びを諦めないための学習機会の提供、教材支援、および市民プロジェクトから生まれた「小さな奨学金」制度の創設を目指します。
Transparency
どこから収益が生まれ、どこへ還ったのか。簡易レポートをGitHub上で定期的に公開し、市民への説明責任を果たします。
